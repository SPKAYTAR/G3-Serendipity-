Detailed Description

Represents a single book record in the Serendipity inventory.

Stores ISBN, title, author, publisher, date added, quantity, wholesale cost, and retail price.

Class Invariants:

  • ISBN is a string up to 13 characters (preserve leading zeros).
  • qtyOnHand ≥ 0
  • wholesale ≥ 0.0
  • retail ≥ 0.0
  • bookCount tracks how many bookType objects exist.

This class provides full getters/setters with validation and is used by the Inventory Database (Add/Edit/Delete).
